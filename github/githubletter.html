A Love Letter to Sweet Ole' Github

Let me start with all the reasons I love you...

You keep me organized. You don't ever hold me down to my first commit.


Speaking of commitment.... if I ever need to go back to the way things were, it's only a matter of a few words and you take me back as if nothing happened in the first place. I can go back, move forward, or stay exactly where I am with you.


You are flexible and dynamic.

I can set up repositories, little databases, by using you. You store all my thoughts, designs, crazy ideas.


First you need to make sure Git is installed on my computer. If not go here.


Once I have everything up and running, I go to Github.

On this site, I can manage all my repositories. I can create new ones, configure existing ones, and delete files if I need to. This platform allows me to open source my code on a timeline. Anyone on the web can visit my code and see what I have been creating and shifting (well, according to your pivacy settings and if you pay for a private account)


With the help of Terminal and Git commands, I gain technical knowledge that will help in any web development projects I encounter.

To see all the commands thus far I have learned about and used please see my Git Love Letter
