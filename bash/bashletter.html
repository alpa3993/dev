A Love Letter to Bad Ole Bash

Bash, you and Terminal open up the world for me.

With your help I can see things in a new way. You give me the power to control my computer in ways I didn't know before. I know I mostly see your shell but in time I hope to know you on a more deeper level.

But before that, lets start with a little ground rules.

  • You are a man of a few characters - 255 to be exact.
  • You mostly only understand A-Z, a-z, 0-9, periods, underscores, and hyphens.
  • Underscores are better with you than spaces... since spaces seem to confuse you.
  • You give me that "stick shift kind of control" that I need in my life.

In the short time I've known you this is what I've come to know...

  • echo "quote" - This makes you pull a Polly want a cracker on me
  • banner -w# "quote" - This blows up your Polly want a cracker to the entered pixel amount.
  • pwd - This allows me to see where I am in my system.
  • ls - This commands you to show me a list of all the items in my current directory.
  • cd "nameofdirectory" - this allows me to switch into a specified directory
  • cd .. - this takes me into the parent directory of my current directory
  • cd ~ - this brings me back to the main directory
  • cd / - this brings me back to the root of my computer
  • mkdir "newfilename" - this will create a new file or directory within my system
  • touch "filename" - this will create a file under whatever name I specify
  • nano "filename" - this allows me to enter into a text editor to write code for the specified file
  • q - this will quit out of all file windows

You are always so helpful, not only do you organize my local files, you can upload them and organize them on the server in just a few short commands.... it might be a little bit of a love triangle between you, our secure shell, and myself... but whatever gets the job done is fine with me.

  • First I open up Terminal and my secure shell, or SSH.
  • Left side will be my local file that I intend to upload while on the right will be our secure shell, or ssh.
  • Type yes
  • Enter password
  • Get into the directory you would like to upload
  • Jump back to local and send to server
  • scp filename alpa3993@creative.colorado.edu:public_html/dev/filenameinfolder
